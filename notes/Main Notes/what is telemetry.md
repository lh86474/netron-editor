## what is telemetry

### Notes

### Telemetry: 
- Automatic collection and transmission of data from remote sources back to a central server
- Usually refers to collecting usage statistics, performance metrics, and error reports to help developers understand how an app is being used and where it is breaking

```
this._telemetry.send('scroll', { percent_scrolled: 90, ... });
```
used to send data back to Google Analytics
