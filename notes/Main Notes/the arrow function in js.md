## the arrow function in js

### Notes

Arrow function: a compact way to write a function expression in javascript
- Remove the function keyword, the return keyword and curly braces

Usually, we will have something like

```
const add = function(a, b) {
  return a + b;
};
```

Arrow function
```
const add = (a, b) => {
  return a + b;
};
```

### The java equivalent of this is called a Lambda Expression
