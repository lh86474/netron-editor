## what is complex64

### Notes

complex64 is a data type to represent **complex numbers**
- Takes 64 bits of memory
- a complex is written in a + bi

(a) is stored as a standard 32-bit Float
(b) is stored as a 32-bit float

The vast majority of NNs just need real numbers, but if we are working with waves, frequencies, or specific physics, we need to use complex64
