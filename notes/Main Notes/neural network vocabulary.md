## neural network vocabulary

### Notes

Batch (N or B): The number of independent images grouped together in a single pass. 
- Let's way a model processes 32 images simultaneously, the batch size is 32

Channels
- The depth of the feature information
	- For a raw input image, this is usually 3 (Red, Green Blue)
	- Deeper inside a network, an operation might output 64 or 128 channels
		- Not colors, but feature maps 

Height
- The vertical spatial dimension, measured in pixels

Width
- The horizontal spatial dimension, measured in pixels
