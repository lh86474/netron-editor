## what is a lexer and parser

### Notes
Lexer (lexical analyzer) often called a tokenizer
- First piece of software when computer tries to read code or structured text
- Take raw string group of chars and group them into **meaningful chunks called token**
- When I read, I automatically group individual letters into words, space as separators, periods as punctuation. A computer doesn't know that 

 Lexer is computer's basic reading comprehension
parser: actually understand the "grammar" of text. Checks upt he syntax is right, and build the final data structure in the computer memory
